# FileMint

FileMint is an iPhone-only SwiftUI app for local file compression and format conversion.

## MVP

- Compress PDF files on device.
- Convert images into a PDF.
- Export PDF pages as JPG images in a ZIP file.
- Convert images between JPEG, PNG, and HEIC.
- Store processed files and local history in the app sandbox.
- Support English and Simplified Chinese UI.
- Provide in-app links to privacy policy and support pages.

## Project

- App name: FileMint
- Chinese display name: 文件薄荷
- Bundle ID: `com.zhouyajie.filemint`
- Version: `1.0.0`
- Platform: iPhone only
- Backend/network: none. Web links open only after a user action.
